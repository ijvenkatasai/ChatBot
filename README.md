AI Smart Chatbot is a Flask web app with an HTML, CSS, and JavaScript frontend connected to Groq AI.
It lets users type questions and get intelligent replies in real time.
Features include chat history, styled interface, and easy setup with Python, making it beginner‑friendly and extendable.

🚀 Features
Flask backend with Groq AI integration

Responsive chat interface (HTML, CSS, JS)

Chat history maintained for smooth conversation

Beginner‑friendly setup and extendable design

🛠️ Tech Stack
Python (Flask, dotenv, groq)

HTML5, CSS3

JavaScript (ES6, Fetch API)
